# homelab-helper-lvm2

This repo provides a custom build of [lvm2](https://www.sourceware.org/lvm2/) for use with the [homelab-helper](https://github.com/benfiola/homelab-helper.git) project.

The build process is triggered by a manual [github workflow](.github/workflows/build-and-release.yaml), running a [build script](./build.sh).
